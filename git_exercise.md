## 简单
- 随意挑选github上任意一个开源项目分别使用http、ssh克隆到本地
- 自行新建一个github开源项目，提交readme.md到github上
- 在本地dev分支上开发，完成10次提交后，合并到本地master分支
- 提交本地dev代码到远程dev分支
- 提交本地master分支到远程master分支
- 从远程dev分支拉取代码到本地
- 需要如何配置使得`git pull, git push`能够默认拉取或提交当前分支
